This project is about implementing a perl library for the Hetzner cloud APIs
as documented on https://docs.hetzner.cloud/

See `perldoc Net::hcloud` for details on usage

## hcloudcli.pl

For `hcloudcli.pl`, you need to install some dependencies with

    zypper in 'perl(Term::ReadLine::Gnu)' 'perl(JSON::XS)' 'perl(YAML)'

Use `help` or tab-completion to see available commands.
